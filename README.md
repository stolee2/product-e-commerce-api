## Installation

```bash
$ npm install
```

```DB
Setup postgres DB on your local machine with name `products-api`
and username `postgres` and password `123456` and run the database on port `5432`
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
